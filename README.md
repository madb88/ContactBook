This is ContactBook Aplication. 
You can create your own contact book. 

Website was created using:

- HTML
- CSS
- Bootstrap for front-end 
- OOP PHP
- Symfony2
- Doctrine 2 ORM, DQL
- Twig templates

Features:

You can add and delete contact
You can edit contacts and add them:
- Description
- Address
- Email address
- Phone number
